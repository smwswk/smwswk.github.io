🔄 同步图库

将 photo/ 文件夹中的新照片同步到网站图库

1

下载同步脚本

点击下方按钮下载 SyncPhotos.command 脚本到本地

⬇️ 下载 SyncPhotos.shortcut

下载后双击即安装到「快捷指令」App

2

运行同步

双击下载的 SyncPhotos.shortcut 文件，系统打开「快捷指令」App 提示安装，确认后即可在任意文件夹右键 → 快速操作 → SyncPhotos 触发同步。

  • 首次运行点「仍要打开」并允许运行
  • 在「系统设置 → 隐私与安全性」确认
  • 脚本自动完成：缩图 → 复制 → 生成索引 → Hugo 构建
3

推送发布

打开 GitHub Desktop → 提交变更 → Push，约 1–2 分钟后 图库 自动更新

⏰ 自动同步

系统已配置每日 08:00 自动同步。如需立即同步，点击上方下载按钮手动触发。
