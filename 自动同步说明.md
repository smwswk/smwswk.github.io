# 📷 照片自动同步使用说明

## 快速开始

### 1. 启动自动监控

在终端运行：
```bash
cd ~/Documents/GitHub/smwswk.github.io
bash start_auto_sync.sh
```

或者直接运行：
```bash
bash watch_photos.sh
```

### 2. 上传照片

**只需要把照片拖到 `photo/` 文件夹**，剩下的全自动：
- ✅ 自动生成缩略图
- ✅ 自动生成 web 版图片
- ✅ 自动构建 Hugo 站点
- ✅ 自动 Git 提交
- ✅ 自动推送到 GitHub

### 3. 停止监控

```bash
bash watch_photos.sh stop
```

---

## 后台运行（推荐）

如果想让监控一直在后台运行，即使关闭终端也不会停止：

```bash
cd ~/Documents/GitHub/smwswk.github.io
nohup bash watch_photos.sh > /dev/null 2>&1 &
```

查看是否在运行：
```bash
ps aux | grep watch_photos
```

停止后台监控：
```bash
bash watch_photos.sh stop
```

## 开机自动启动（可选）

如果想让电脑开机后自动启动监控，可以创建一个 LaunchAgent：

1. 创建配置文件：
```bash
nano ~/Library/LaunchAgents/com.smwswk.photo-sync.plist
```

2. 粘贴以下内容（注意修改路径）：
```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.smwswk.photo-sync</string>
    <key>ProgramArguments</key>
    <array>
        <string>/bin/bash</string>
        <string>/Users/sunminwen/Documents/GitHub/smwswk.github.io/watch_photos.sh</string>
    </array>
    <key>RunAtLoad</key>
    <true/>
    <key>KeepAlive</key>
    <true/>
    <key>StandardOutPath</key>
    <string>/Users/sunminwen/Documents/GitHub/smwswk.github.io/.watch_photos.log</string>
    <key>StandardErrorPath</key>
    <string>/Users/sunminwen/Documents/GitHub/smwswk.github.io/.watch_photos.log</string>
</dict>
</plist>
```

3. 加载配置：
```bash
launchctl load ~/Library/LaunchAgents/com.smwswk.photo-sync.plist
```

4. 取消开机启动：
```bash
launchctl unload ~/Library/LaunchAgents/com.smwswk.photo-sync.plist
```


## 工作原理

1. `watch_photos.sh` 使用 `fswatch` 监控 `photo/` 文件夹
2. 检测到新图片（jpg/jpeg/png/webp/tiff）时，自动运行 `sync_photos.sh`
3. `sync_photos.sh` 处理图片、构建站点、提交并推送到 GitHub

---

## 故障排除

### 监控没有启动
- 检查 `fswatch` 是否安装：`brew install fswatch`
- 查看日志：`cat .watch_photos.log`

### 图片没有自动上传
- 确认监控正在运行：`ps aux | grep watch_photos`
- 查看日志：`tail -f .watch_photos.log`
- 手动运行一次：`bash sync_photos.sh`

### Git 推送失败
- 检查 SSH 配置：`ssh -T git@github.com`
- 查看 Git 状态：`git status`
